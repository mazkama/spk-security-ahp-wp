# Kesimpulan

Aplikasi SPK rekrutmen petugas keamanan berbasis web berhasil dibangun menggunakan Laravel 10, MySQL, dan Blade. Metode AHP dan WP berjalan baik untuk menghitung bobot kriteria dan perangkingan kandidat. Sistem membantu pengambilan keputusan rekrutmen secara objektif, transparan, dan efisien.
