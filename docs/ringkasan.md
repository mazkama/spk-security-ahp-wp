# Ringkasan Sistem

Aplikasi ini adalah Sistem Pendukung Keputusan (SPK) berbasis web untuk rekrutmen petugas keamanan. Sistem menggunakan metode Analytical Hierarchy Process (AHP) untuk menghitung bobot kriteria dan Weighted Product (WP) untuk perangkingan kandidat.

- **Tujuan:** Membantu proses seleksi secara objektif dan transparan.
- **Metode:** AHP (bobot kriteria), WP (ranking kandidat)
- **Teknologi:** Laravel 10, MySQL, Blade
- **Konsep:** Berbasis periode, semua data terikat ke periode seleksi.
