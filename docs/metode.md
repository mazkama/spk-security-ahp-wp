# Penjelasan Metode

## AHP (Analytical Hierarchy Process)
- Pairwise comparison antar kriteria
- Normalisasi matriks
- Perhitungan bobot kriteria
- Consistency Ratio (CR) untuk validasi konsistensi

## WP (Weighted Product)
- Perkalian berbobot nilai penilaian
- Perhitungan nilai S kandidat
- Perhitungan nilai V (normalisasi)
- Proses ranking berdasarkan nilai V
