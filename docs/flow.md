# Flow Sistem

1. **Login**: User melakukan autentikasi.
2. **Periode**: Setiap proses seleksi terikat pada satu periode.
3. **Kriteria**: Admin mendefinisikan kriteria seleksi.
4. **AHP**: Pairwise comparison antar kriteria untuk bobot.
5. **Kandidat**: Admin menambah kandidat.
6. **Penilaian**: Kandidat dinilai berdasarkan kriteria.
7. **WP**: Sistem menghitung ranking kandidat.
8. **Laporan**: Hasil ranking dan ekspor laporan.

Setiap tahap dijalankan secara berurutan dan terintegrasi.
